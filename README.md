# AdventOfCode2025
## How to Run:
Compile each file with:     
```gcc -lm -o [name] Advent#.c```     
Enter input through STDIN, after that, send signal SIGINT (CTRL+C) to stop the program and output the final answer.      
Each program is set up by default to compute Part 2, uncomment the part 1 section of code to run part 1 instead.
